# 🐾 Patas - Plataforma de Adoção Animal

## 📌 Sobre o Projeto
O Patas é um projeto desenvolvido com o objetivo de incentivar a adoção responsável de animais.  
A plataforma permite que usuários visualizem animais disponíveis para adoção, conheçam informações sobre cada pet e entrem em contato para realizar o processo de adoção.

O projeto foi desenvolvido como atividade acadêmica/prática para aprimorar conhecimentos em desenvolvimento web.

---

## 🚀 Tecnologias Utilizadas

- HTML
- VScode
- JS
- Git e GitHub

---

## 📖 Funcionalidades
- Sistema de navegação entre páginas
- Área de adoção de animais
- Formulário de adoção
- Área de contato
- Header e Footer personalizados
